# 🏥 Health Referral System - Web Admin Panel

A web-based administration platform developed as part of a healthcare innovation hackathon to help hospitals manage critical healthcare resources and improve referral coordination between healthcare providers.

---

## 📌 Overview

The Health Referral System Admin Panel enables healthcare administrators to monitor and manage hospital resources such as ICU beds, ventilators, and general bed availability through a centralized dashboard.

The platform aims to improve visibility into hospital capacity and support more efficient healthcare referral workflows.

---

## 🏆 Hackathon Project

This project was developed during a healthcare-focused hackathon.

### Problem Statement

Hospitals often struggle with fragmented information regarding resource availability, making patient referrals and emergency coordination difficult.

### Solution

The Health Referral System provides a centralized administration dashboard that allows healthcare organizations to manage and update resource availability in real time, helping improve decision-making and referral processes.
