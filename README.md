# 💻 Health Referral System - Web Admin Panel

## 📌 Overview
This is the web-based admin panel for the Health Referral System. It allows hospitals or administrators to manage and update hospital resource data such as ICU beds, ventilators, and availability.

---

## 🚀 Features
- Hospital resource management
- Update ICU, beds, ventilator availability
- Admin dashboard interface
- Real-time data control (for future integration)

---

## 🛠️ Tech Stack
- Frontend: React / HTML / CSS / JavaScript
- API Integration: Backend APIs (Spring Boot / FastAPI)

---

## ⚙️ Functionality
- Add / Update hospital details
- Manage resource availability
- View system data (optional)

---

## 🔄 Current Status
- UI: ✅ Completed
- Backend Integration: ⚠️ In progress
- Ready for real-time data integration
